Welcome on POD home page.

POD is a Python tool that allows you to easily create FPGA bitfiles for your embedded system, from several Open Source Virtual Components (compatibles with the OpenCores Wishbone bus). It will also generates the corresponding drivers (currently only ARMadeus Linux ones).

To get some help or to discuss about POD, join us on IRC (irc://freenode/#pod) or subscribe on mailing list.
